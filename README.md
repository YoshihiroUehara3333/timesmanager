# timesmanager
times投稿を管理したい

## 機能
- timesチャンネル内でbotにメンションを付けて投稿された日報をDBに保存する
- 投稿された日報が編集されたときにDBのレコードを上書きする
- 投稿のスレッド内でメンション時、chatGPTによるフィードバックを生成しスレッドに返信する

#### 日報投稿フォーマット
【日付】0000-00-00 <br>
【時間】09:00～18:00 <br>
【業務内容】<br>
業務内容1<br>
業務内容2<br>
<br>
【自己評価】<br>
うまくいった点<br>
改善できる点<br>
<br>
【翌日の計画】<br>
予定している作業と完了目標<br>
懸念点や準備しておくべきこと<br>
 <br>
【その他】<br>
その他の内容
